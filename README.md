# schedulebuilder

## About

A web application for building KU schedules created by Addison, Anya, Elizabeth, Hunter, Janna, and Marco for EECS 581: Software Engineering II. \
This project was implemented with NextJS, Tailwind, NextAuth and Supabase. 

## Use / Install

This project might be accessible at https://schedulebuilder.addiebart.net but it also might not depending upon when you're reading this.

Unfortunately because the project hinges on server access to a database, it won't be usable if cloned.
